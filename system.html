SYSTEM OVERVIEW

The Big Picture

KAMMAR SODA is organized into four software layers that connect the simulated production process, Python control logic, HMI monitoring, and web-based project documentation.

Physical Layer

Represents the six production stations: Bottle Blow, Soda Fill, Carbonation, Cap On, Label, and QC & Pack.

Control Layer

Python logic controls station flow, production counters, state changes, defect detection, and alarm behaviour.

Data Layer

Production values are prepared for monitoring and can be connected to InfluxDB and Grafana for industrial-style dashboards.

Visualization Layer

The HMI and website provide operators and viewers with clear system visibility, process explanation, and technical documentation.

System Architecture

SEMI E10 Time Buckets

SEMI E10 classifies equipment time into productive, standby, engineering, scheduled downtime, unscheduled downtime, and non-scheduled states. In this project, it is used to explain uptime, downtime, and production monitoring for the KAMMAR SODA line.
