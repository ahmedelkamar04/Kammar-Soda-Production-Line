System Overview

The SparkFlow Soda Plant is organized into four layers. The physical layer models the six manufacturing stations. The Python backend controls bottle movement, state transitions, production counters, and defect detection.

The HMI provides operator controls and live process monitoring. Production data is stored in InfluxDB and visualized using Grafana dashboards.

Operator HMI

The Human Machine Interface (HMI) is the primary operator control panel for the SparkFlow Soda Plant. It provides real-time visibility of production performance, bottle counts, machine status and alarm conditions.

Operators can start, stop, reset and safely shut down the production line using dedicated control buttons. The HMI also displays the active production stage and overall system efficiency, allowing rapid response to faults and process interruptions.
